# Security policy

Security vulnerabilities for Neuro MCP Relay & Registry are handled separately from VSC-NeuroPilot.
Please open a private security report on GitHub and fill out all the details.

Do not open an issue or discussion thread regarding the vulnerability, as this will publicly reveal the vulnerability.

## View access

Only those working directly on Neuro MCP Relay & Registry will be able to read the reports. No one from the VSC-NeuroPilot team will be able to read the reports unless they also regularly work on this project.
